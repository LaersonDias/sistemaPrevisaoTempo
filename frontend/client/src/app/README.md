# Sistema de Previsão do Tempo

Projeto desenvolvido para a disciplina
Engenharia de Software II - ISPTEC.

## Tecnologias

- Angular
- PHP
- MySQL
- OpenWeather API

## Funcionalidades

- Login e registo
- Pesquisa de cidades
- Dark mode
- Multi-idioma
- Exportação PDF/CSV
- Favoritos
- Histórico
- Responsividade

## Execução

Frontend:

npm install
ng serve

Backend:
Colocar backend em htdocs/XAMPP

Base de dados:
Importar script SQL
